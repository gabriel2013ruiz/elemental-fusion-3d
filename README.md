# ⚡ Elemental Fusion 3D · 20 Levels

A **first-person** stealth game built with a pure-JavaScript **raycasting engine** (Wolfenstein-style 3D, no libraries).
A powerful **Seeker** hunts you through a maze. You start holding **🔥 Fire** — find the other elements,
then **FUSE** them all to destroy the Seeker. Survive **20 levels**, each adding new features.

## 🎮 How to play
- You can't beat the Seeker head-on. Carry **🔥 Fire** from the start.
- **Find the other element orbs** (💧 Water, 🪨 Earth, 🌪️ Wind) hidden in the maze.
- Press **`F`** to **FUSE** all your powers into one blast that destroys the Seeker(s) and clears the level.
- When a Seeker gets close, press **`SPACE`** to **FREEZE** every Seeker for **8 seconds** — your escape move (limited charges).

## ⌨️ Controls
| Action | Keys |
|--------|------|
| Move / strafe | `W` `A` `S` `D` |
| Look | `←` `→` or **mouse** (click to lock) |
| Sprint | `Shift` |
| Freeze (8s) | `Space` |
| Fuse | `F` |
| Toggle map size | `M` |
| Pause / help | `P` |

**On phone:** drag the **left** side to move, drag the **right** side to look, and use the **Freeze** / **Fuse** buttons.

## 📈 The 20 levels
Each level adds something new: bigger mazes, more elements, fog/limited vision, more Seekers
(up to **five**), raging Seekers that sprint when they spot you, fewer freeze charges… ending with the
**Mega-Seeker** on level 20.

## 🚀 Run locally
Open `index.html` in any browser.
